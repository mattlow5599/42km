What "run-walk" means — and how you run more over time

Jog a set time, walk a set time, repeat. The ratio is run : walk in minutes.

4:1 = jog 4 min, walk 1 min. (Dark = jog, yellow = walk.)
  • Run blocks lengthen every phase (3:1 → 10:1) so you end up running most of it.
  • Walk briskly; start breaks from minute one, even when fresh.
  • From ~Week 14, easy runs can be continuous if shins are happy — ease back the moment they complain.

Run easy — the habit that makes or breaks this
  • Easy = conversational. Full sentences. Can't? Slow down or walk. ~80% of running should feel this easy.
  • It'll feel too slow — roughly 7:30–8:00/km for now, not low-6s.
  • Even, not fading. Finish each run as strong as you start. Fading = went out too hard.
  • HR climbing toward max on an easy run = too hard.
  • Why: easy running builds the engine that lets you run more & walk less on race day.
Before & after every run
Warm-up
~5 min · never skip
  • 3 min brisk walk.
  • Leg swings ×10 each; ankle circles ×10.
  • 6 lunges · high knees · butt kicks.
  • Gentle calf + toe raises (shins).
  • No static stretching cold.
Cool-down
~5–10 min
  • 3–5 min easy walk.
  • Stretch 20–30s: calves, quads, hams, hips, glutes.
  • Foam-roll calves; massage shins.
  • Rehydrate; carbs + protein after long runs.
  • Shins hot? Ice + elevate.

Shin survival kit

  • Cadence ~170–180/min — quick, light, short steps.
  • If shins grumble: trim weekday distance first, keep the long run, repeat the week.
  • Soft over hard surfaces; proper shoe fitting; replace ~600–800km.
  • Golden rule: dull ache that warms up = OK to jog gently. Sharp pinpoint bone pain = STOP, rest, see a physio.

Strength — Wed + Sun, ~15 min

  • Tibialis raises 3×15 (#1 shin-splint preventer) · eccentric calf raises 3×12.
  • Glute bridges 3×10 · clamshells 3×12 · split squats 3×8 · planks.

Singapore heat & fuel

  • Run before ~7:30am or after ~7pm; go by effort, not pace.
  • Hydrate all day; electrolytes for runs over ~45–60 min.
  • Long runs >75–90 min: 30–60g carbs/hour; never try anything new on race day.

Race day — run more, walk less (6 Dec)

  • Run the route, walk the aid stations (~30–45s each) → ~4–5km walking, under your 7km.
  • Target ~9:00–9:15/km → ~6:30. Cut-off 7:00. Register before 30 Sep.
  • Fading? Drop to 6:1 run-walk — finishing strong beats chasing zero walking.

Head game

  • Chunk it — only think to the next walk break. Mantra on hard patches.
  • 5-minute rule: never quit while you feel awful — walk, drink, reassess after 5 min.
  • Push through discomfort, never pain. Walk breaks are a tool, not quitting.

Coaching plan, not medical advice. With your shin history, a GP/physio check before ramping up is wise.
