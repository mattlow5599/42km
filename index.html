Racerun + aid walks
Run easy — the make-or-break habit
  • Easy = conversational. Full sentences. ~80% of running this easy.
  • Your easy pace ≈ 7:45–8:15/km right now, not low-6s. It'll feel too slow — good.
  • Even, not fading. Finish as strong as you start.
  • HR near max on an easy run = too hard.
  • Why: easy builds the engine to run more & walk less on race day.
Before & after every run
Warm-up
~5 min
  • 3 min brisk walk.
  • Leg swings, ankle circles.
  • Lunges, high knees, butt kicks.
  • Calf + toe raises (shins).
Cool-down
~5–10 min
  • Easy walk down.
  • Stretch calves, quads, hams, hips.
  • Foam-roll; massage shins.
  • Refuel; ice shins if hot.

Shin survival

  • Cadence ~170–180/min — short, light steps.
  • Grumble? Trim weekday distance first, keep the long run, repeat the week.
  • Dull ache that warms up = OK. Sharp bone pain = STOP, see a physio.

Heat & fuel (Singapore)

  • Run before ~7:30am / after ~7pm. Effort, not pace.
  • Hydrate; electrolytes >45–60 min.
  • Long runs >75–90 min: 30–60g carbs/hr; nothing new on race day.

Race day (6 Dec)

  • Run the route, walk the aid stations (~30s each) → ~4–5km walking.
  • ~9:00–9:15/km → ~6:30. Cut-off 7:00. Register before 30 Sep.

Head game

  • Chunk it to the next walk break. Mantra on hard patches.
  • 5-min rule: never quit while you feel awful — walk, drink, reassess.
  • Push through discomfort, never pain.

Coaching guidance, not medical advice. With your shin history, a GP/physio check before ramping up is wise.
